# YJ100KMob

The `yjmob100k-dataset1_sample.csv` contains 1000 rows as a sample from the YJ100KMob dataset 1, available from [Zenodo](https://zenodo.org/records/10836269).

The `activity.csv`, the `unique_users.csv`, and the `grid_bl_2449.geojson` were published in [GitHub](https://github.com/pintergreg/reverse-engineering-YJMob100K-grid).
